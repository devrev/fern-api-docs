<Callout intent="Security: API key">
All the DevRev APIs require a token to authenticate the user. Provide `Authorization: <token>` as a header to every API request. [How do I find my token?](/welcome/authentication)

An API key is a token that you provide when making API calls. Include the token in a header parameter called Authorization.

Example: `Authorization: 123`
</Callout>
