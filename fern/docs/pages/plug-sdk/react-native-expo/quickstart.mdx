# Quickstart Guide

This guide helps you integrate the DevRev SDK into your React Native Expo app.

## Requirements

- Expo SDK 44 or higher.
- React Native 0.64 or later.

## Installation

To install the DevRev SDK, run the following command:

```sh
expo install @devrev/sdk-react-native
```

## Set up the DevRev SDK

1. Open the DevRev web app at [https://app.devrev.ai](https://app.devrev.ai) and go to the **Settings** page.
2. Under **PLuG Chat**, copy the value under **Your unique App ID**.
3. After obtaining the credentials, configure the DevRev SDK in your app.

<Callout intent="note">
The DevRev SDK must be configured before you can use any of its features.
</Callout>

The SDK will be ready for use once you execute the following configuration method:

```typescript
DevRev.configure(appID: string)
```
