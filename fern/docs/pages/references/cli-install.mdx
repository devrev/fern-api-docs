The DevRev CLI is a command-line interface tool that simplifies working with the DevRev REST APIs.


<Callout intent="note">
    Uninstall the previous version of the DevRev CLI before installing the latest version.
    <Tabs>
        <Tab title='Linux'>
            ```bash
            sudo dpkg -r devrev
            ```
        </Tab>
        <Tab title='Mac'>
            ```bash
            brew uninstall devrev
            ```
        </Tab>
        <Tab title='Windows'>
            1. Remove the path from the environment variable. For example: `C:\Program Files\devrev\`
            2. Delete the downloaded executable file and its folder.
        </Tab>
    </Tabs>
</Callout>

## Install the DevRev CLI using NPM

You can install the DevRev CLI using npm (Node Package Manager). Run the following command:
```bash
npm install -g devrev
```
This command installs the DevRev CLI globally on your system, allowing you to use it from any directory.

After installation, you can verify the installation by checking the version of the DevRev CLI:
```bash
devrev --version
```
This command displays the installed version of the DevRev CLI, confirming that it has been installed successfully.
