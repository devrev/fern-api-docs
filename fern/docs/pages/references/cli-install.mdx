The DevRev CLI is a command-line interface tool that simplifies working with the DevRev REST APIs.


<Callout intent="note">
    The installation of the DevRev CLI via Homebrew (for MAC), dpkg (for Linux), or manual methods (for Windows and Mac) is no longer supported.

    If you have previously installed the DevRev CLI using these deprecated methods, please uninstall the earlier version.
</Callout>
<Tabs>
    <Tab title='Linux'>
        ```bash
        sudo dpkg -r devrev
        ```
    </Tab>
    <Tab title='Mac'>
        ```bash
        brew uninstall devrev
        ```
    </Tab>
    <Tab title='Windows'>
        1. Remove the path from the environment variable. For example: `C:\Program Files\devrev\`.
        2. Delete the downloaded executable file and its folder.
    </Tab>
</Tabs>

## Install the DevRev CLI using NPM

To install the DevRev CLI, please utilize npm (Node Package Manager) by executing the following command:
```bash
npm install -g devrev
```
This command facilitates the global installation of the DevRev CLI on your system, enabling its use from any directory.

Upon completion of the installation, confirm its success by checking the version of the DevRev CLI.
```bash
devrev --version
```
This command presents the currently installed version of the DevRev CLI, thereby verifying its successful installation.
