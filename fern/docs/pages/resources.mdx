## Prerequisites

- Create a [dev org](https://app.devrev.ai/)
- Install [DevRev CLI](https://github.com/devrev/cli)
- Install [jq](https://stedolan.github.io/jq)
- Get [DevRev SDK](https://www.npmjs.com/package/@devrev/typescript-sdk)

## Creating DevRev conversations

To create a new conversation for a dev org, do the following:

1. In the DevRev webapp go to **Settings** > **PLuG Settings**.
2. To set up a PLuG widget for the first time, click **+ PLuG Widget**. Otherwise, proceed to the next step.
3. Click **View in Playground** to open the PLuG widget.
4. Click **Send us a message** and send a message.
